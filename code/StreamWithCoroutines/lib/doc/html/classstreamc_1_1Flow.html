T& streamc::Flow::createOperator ( std::string const &  name,
Args &&...  args 
)
inline

Create an operator and add it to the flow.

This function instantiates an operator and adds it it the flow. It is the preferred way to add operator's to a flow. Example:

Operator & op = flow.createOperator<FileSource>("op-name")
.set_fileName("in.data")
.set_fileFormat({{"name", Type::String}, {"age", Type::Integer}});

or alternatively:

Operator & op = flow.createOperator<FileSource>("op-name",
"in.data", {{"name", Type::String}, {"age", Type::Integer}});
